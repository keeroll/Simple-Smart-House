# Simple-Smart-House

### Hardvare
- Arduino Uno Rev 3
- Temperature and humidity sensor (DHT 11)
- RFID sensor (RC 522)
- Infrared motion sensor (HC-SR501)
- Light sensor (LDR)
- Servomotor (MG 90s Micro)
- Wires, resistors, LEDs, etc.
  
### Software
As monitoring and managemet software I used Blynk. It's the mobile app for IoT.
